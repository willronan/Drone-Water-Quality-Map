Live Dashboard found at https://willronan.github.io/Drone-Water-Quality-Map/index.html

Data Schema:
Fields:

- dateKey = YYYY-MM-DD
- dateLabel = e.g. April 20
- timestamp = YYYY-MM-DDTHH:MM:SSZ
- EnclosureTime = HH:MM:SSZ
- latitude 
- longitude
- salinity = measured from Atlas Conductivity probe in PSU 
- temperature = measured from Blue Robtoics Celcius temp sensor
- altitude  = as measured from Blue Robotics Bar Pressure Sensor
- pressure = as measured from Blue Robotics Bar Pressure Sensor
- tempPressureSensor = as measured from Blue Robotics Bar Pressure Sensor
- deviceType = e.g. "Drone"
- deviceId = ID of specific device 
- text = any notes related to sample
- error = any device-side errors at time of sample
